# FRO Bokstavering

Träningsspel för att öva bokstavering enligt [FRO](https://fro.se) (Frivilliga Radioorganisationen).

Baserat på minneskort Bokstavering V1 från [minneskort.froverktyg.se](https://minneskort.froverktyg.se).

## Funktioner

- **Svenska och engelska** — växla språk när som helst
- **Tre speltyper** — blandat, bara alfabet, eller siffror & skiljetecken
- **Två lägen** — alla tecken eller ett begränsat urval
- **Tidtagning** — total tid och snitt per tecken
- **Resultatöversikt** — se vilka tecken du svarade fel på med rätt svar
- **Facit** — fullständig referenstabell för alla tecken

## Kom igång

Öppna `index.html` i en webbläsare. Inga beroenden eller byggsteg krävs.

## Data

All bokstaveringsdata kommer från FRO:s minneskort och inkluderar:

- **30 bokstäver** (A–Ö samt Ü)
- **10 siffror** (0–9)
- **19 skiljetecken** (punkt, komma, snedstreck, m.fl.)
